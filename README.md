# Iris_dashboard
## Description

This project is a Machine Learning dashboard developed using the Iris dataset.
The application predicts Iris flower species based on flower measurements using a Random Forest Classifier model.
The dashboard was built with Streamlit and allows users to interactively input flower measurements and receive predictions in real time.

## Features
- Iris species prediction
- Interactive sliders for flower measurements
- Machine Learning classification model
- Accuracy, Precision, Recall, and F1 Score metrics
- Species distribution visualization
- Scatterplot visualization
- 3D interactive visualization
- Dataset preview table
  
## Technologies Used
- Python
- Pandas
- Streamlit
- Matplotlib
- Seaborn
- Scikit-learn
- Plotly

## Machine Learning Model
The project uses: RandomForestClassifier

to classify Iris flower species into:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

## Files Included

app.py: Main Streamlit dashboard 
Iris.csv Dataset: used in the project
requirements.txt: Python dependencies 
README.md: Project documentation 
analysis.ipynb:  Exploratory data analysis notebook 

## How to Run the Project

### 1. Clone the repository

```
git@github.com:Garu84/Iris_dashboard.git
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run Streamlit

```
streamlit run app.py
```
